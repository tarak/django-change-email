Signals¶

django-change-email provides three django.dispatch.Signal classes:

email_change_confirmed¶

A signal that is sent when an email address change request is confirmed. Receives a Request object as single providing argument.

email_change_created¶

A signal that is sent when an email address change request is created. Receives a Request object as single providing argument.

email_change_deleted¶

A signal that is sent when an email address change request is deleted. Receives a Request object as single providing argument.
