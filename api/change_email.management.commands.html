Management commands¶

django-change-email ships one management command that handles the expiration of email address change requests:

cleanupemailchangerequests¶

class change_email.management.commands.cleanupemailchangerequests.Command¶

Bases: django.core.management.base.NoArgsCommand

The cleanupemailchangerequests command deletes expired email address change requests from the database.

Usage:

$ python manage.py cleanupemailchangerequests
